# boxSdf

SDF for a box, optionally infinite one one axis.

## Parameters

* Box Type (`Boxtype`)
* Infinite Axis (`Infiniteaxis`)
* Translate (`Translate`)
* Scale (`Scale`)
* Uniform Scale (`Uniformscale`)
* Inspect (`Inspect`)
