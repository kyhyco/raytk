# waveFn

A function that uses a periodic wave, with the position as the parameter.

## Parameters

* Wave (`Function`)
* Coord Type (`Coordtype`)
* Context Type (`Contexttype`)
* Period (`Period`)
* Phase (`Phase`)
* Amplitude (`Amplitude`)
* Offset (`Offset`)
* Inspect (`Inspect`)
