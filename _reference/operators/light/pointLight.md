# pointLight

Light eminating from a single point in space, with optional distance attentuation.

## Parameters

* Position (`Position`)
* Intensity (`Intensity`)
* Color (`Color`)
* Attenuated (`Enableattenuation`)
* Attenuation Start (`Attenuationstart`)
* Attenuation End (`Attenuationend`)
* Inspect (`Inspect`)
