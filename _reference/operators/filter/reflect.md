# reflect

Reflects space across a plane.

Can optionally expose which side of the plane a point is on as an iteration value for upstream ops.

## Parameters

* Enable (`Enable`)
* Direction (`Direction`)
* Plane Normal (`Planenormal`)
* Offset (`Offset`)
* Shift (`Shift`)
* Expose Iteration (`Exposeiteration`)
* Inspect (`Inspect`)
