# noiseField

A float or vector field that uses one of several noise functions.

## Parameters

* Noise Type (`Noisetype`)
* Coord Type (`Coordtype`)
* Context Type (`Contexttype`)
* Axis (`Axis`)
* Translate (`Translate`)
* Scale (`Scale`)
* Inspect (`Inspect`)
