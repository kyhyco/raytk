# axisDistanceField

A float field that provides the distance from a specific point along a single axis.

## Parameters

* Axis (`Axis`)
* Center (`Center`)
* Coord Type (`Coordtype`)
* Context Type (`Contexttype`)
* Inspect (`Inspect`)
