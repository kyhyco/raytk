# colorRampField

A vector field that maps an input field to values from a range of colors.

## Parameters

* Enable (`Enable`)
* Color 1 (`Color1`)
* Color 2 (`Color2`)
* Alpha 1 (`Alpha1`)
* Alpha 2 (`Alpha2`)
* Clamp Range (`Clamprange`)
* Inspect (`Inspect`)
