# textureField

A float or vector field that looks up values from a texture.

## Parameters

* Coord Type (`Coordtype`)
* Return Type (`Returntype`)
* Context Type (`Contexttype`)
* Plane (`Axis`)
* Translate (`Translate`)
* Scale (`Scale`)
* Texture (`Texture`)
* Extend Mode (`Extendmode`)
* Inspect (`Inspect`)
